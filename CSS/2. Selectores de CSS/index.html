Selectores CSS

Aplicando CSS a diferentes partes de HTML

1. El selector de elementos selecciona elementos basados en el nombre de la etiqueta HTML.

  1. Los selectores de clase seleccionan elementos basados en el valor del atributo class.
  2. Los selectores de ID seleccionan elementos basados en el valor del atributo id.
  3. Los selectores de atributo seleccionan elementos basados en sus atributos y valores.
  4. El selector universal selecciona todos los elementos.
